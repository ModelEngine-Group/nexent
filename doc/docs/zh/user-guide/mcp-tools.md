# MCP 工具

即将推出的 MCP 工具管理模块将让您在一个页面集中管理 MCP 服务器与工具，轻松完成连接配置、工具同步和健康状态监控

## 🎯 功能预览

1. 注册并管理多个 MCP 服务器
2. 快速同步、查看并整理 MCP 工具列表
3. 实时监控 MCP 连接状态和使用情况

## ⏳ 敬请期待

MCP 工具管理功能正在开发中，我们致力于打造一个高效、直观的管理平台，让您能够：

1. 集中管理所有 MCP 服务器
2. 便捷同步和组织工具
3. 实时掌握服务器连接与工具运行状态

## 🚀 相关功能

在等待 **MCP 工具** 上线期间，您可以：
