# 监控与运维

即将推出的监控与运维中心将为智能体提供统一的管理平台，让您实时跟踪健康状态、性能指标与异常事件

## 🎯 功能预览

1. 实时监控智能体健康状态、延迟与错误率
2. 查看并筛选运行日志和历史任务
3. 配置告警策略与关键事件的运维操作
